Comau Dual Arm Robot

Parts Designed

Custom Motherboard Holder

It is designed to be mounted on aluminum profile extrusion frames and to fit both ATX and micro-ATX Motherboards.

It consists of:
•   1 x Holder for the I/O.
•   3 x Corner Holders.
•   4 x Raisers.

The parts were designed using CATIA V5 and 3D printed using a commercial printer.

Custom Gripper

Using the custom Flange, a pneumatic gripper is attached at the end of the robot's arm.

The custom Jaws for the gripper, can hold RAM sticks with or without heat sinks.

Also designed using CATIA V5 and 3D printed using a commercial printer.

Programming the Robot

At first, the robot was manually controlled using the robot pendant. This was done in order to prove that the insertion of a RAM stick to the corresponding RAM slot on the Motherboard was possible. Below you can see a video of that attempt.

Then the robot was programmed to follow the path of picking up the RAM, inserting it into the Motherboard and returning to the initial position. This was achieved using the robot pendant and the PDL2 Programming Language of Comau. Below you can see a video of that attempt.
